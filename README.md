# 🐼 Learning Pandas

A hands-on repository for learning the [pandas](https://pandas.pydata.org/) library — Python's go-to tool for data manipulation and analysis.

---

## 📚 What's Inside

| Topic | Description |
|---|---|
| `01_series/` | Creating and working with Series objects |
| `02_dataframes/` | DataFrame basics — creation, inspection, and structure |
| `03_indexing/` | loc, iloc, boolean indexing, and fancy indexing |
| `04_cleaning/` | Handling missing values, duplicates, and type casting |
| `05_aggregation/` | groupby, pivot tables, and aggregation functions |
| `06_merging/` | merge, join, and concat |
| `07_visualization/` | Plotting directly with pandas |
| `08_real_world/` | End-to-end mini projects on real datasets |


## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- pip or conda

### Installation

```bash
# Clone the repo
git clone https://github.com/your-username/learning-pandas.git
cd learning-pandas

# Create a virtual environment (recommended)
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### Running Notebooks

```bash
jupyter notebook
```

Then open any `.ipynb` file in the browser to follow along.

## 🗺️ Learning Path

If you're just starting out, follow the notebooks in order:

1. **Series & DataFrames** — understand the core data structures
2. **Indexing & Slicing** — learn to select exactly the data you want
3. **Data Cleaning** — handle messy, real-world data
4. **Grouping & Aggregation** — summarize and analyze data
5. **Merging Datasets** — combine multiple data sources
6. **Visualization** — turn data into charts quickly
7. **Real-world Projects** — put it all together
